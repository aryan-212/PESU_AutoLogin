## Installation

1. For bash:
   ```bash
   sudo curl -sL https://github.com/aryan-212/lobster-wsl/raw/main/login.sh -o /usr/local/bin/pesu && sudo chmod +x /usr/local/bin/pesu
  
2. For windows:
   ```powershell
   git clone https://github.com/aryan-212/PESU_AutoLogin.git
   cd PESU_AutoLogin
   .\autoLog.ps1
